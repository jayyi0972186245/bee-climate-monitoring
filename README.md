# 智慧蜂箱特殊氣候監測平台

## 專案介紹

本 Repository 整合兩個期末作業成果：

1. **台灣特殊氣候蜜蜂監測分析 Skill**
2. **智慧蜂箱特殊氣候監測網站**

作品以台灣各地特殊氣候為基礎，結合即時天氣資料，分析不同地區蜂箱可能面臨的環境風險。使用者選擇縣市後，網站會顯示該地特殊氣候、即時溫度、即時風速、即時濕度、蜜蜂影響、風險等級與監測建議。

網站左側加入 Google 地圖蜂場連動面板，會依照所選地區顯示對應蜂場搜尋結果；其中屏東以「陳家蜂蜜」作為範例。

- 新增過去 24 小時歷史趨勢圖，顯示溫度、濕度與風速折線
- 分析蜜蜂活動與蜂箱環境風險
- 提供智慧蜂箱監測與控制建議

## 網站功能

1. 選擇縣市
2. 顯示地區特殊氣候
3. 取得即時溫度、風速、濕度
4. 顯示 Google 地圖蜂場查詢
5. 左側嵌入 Google 地圖，並依縣市連動更新
6. 顯示蜜蜂活動風險指數：正常、注意、高風險
7. 將高溫、高濕、強風與降雨轉成可執行建議
8. 使用蜂箱 AI 建議助手產生蜂農建議文字
9. 顯示過去 24 小時溫度、濕度、風速趨勢
10. 顯示蜜蜂可能受到的影響
11. 顯示蜂箱風險等級
12. 顯示監測建議與控制設備

## Skill 工作流程

本專案的 Skill 是「台灣特殊氣候蜜蜂監測分析 Skill」，主要流程如下：

1. 收集各地氣候特色
2. 分析氣候對蜜蜂活動的影響
3. 轉換成蜂箱監測規則
4. 判斷風險等級
5. 產生網站可顯示的監測建議文字

## 即時天氣資料

本作品使用 Open-Meteo API 取得即時資料：

- `temperature_2m`：即時溫度
- `relative_humidity_2m`：即時相對濕度
- `wind_speed_10m`：即時 10 公尺風速
- `hourly`：過去 24 小時溫度、濕度與風速趨勢資料

## 風險判斷概念

- 高溫：啟動霧化器、遮陰與溫度警報
- 高濕：加強防潮與病害監測
- 強風：降低蜜蜂外勤活動判斷，加入風速警示
- 低溫：提醒蜂箱保溫並減少開箱
- 降雨：降低外出採蜜預期，避免開箱檢查
