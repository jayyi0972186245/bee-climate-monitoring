# 台灣特殊氣候蜜蜂監測分析 Skill

## 專案介紹

本 Skill 用於協助智慧蜂箱監測網站分析台灣各地特殊氣候與即時天氣資料對蜜蜂活動的影響，並產生對應的監測建議。

## Skill 目的

協助使用者完成以下流程：

1. 輸入縣市名稱
2. 查找該地特殊氣候與代表座標
3. 取得或輸入即時溫度、風速、濕度
4. 分析可能影響蜜蜂活動的環境因素
5. 判斷蜂箱風險等級
6. 輸出智慧蜂箱監測建議
7. 產生 Google 地圖蜂場查詢連結
8. 產生蜜蜂活動風險指數與蜂農可執行建議

## 輸入

- 縣市名稱
- 溫度
- 濕度
- 風速
- 特殊氣候

## 輸出

- 蜜蜂活動狀態
- 蜂箱風險等級
- 即時天氣數據
- Google 地圖蜂場查詢
- 蜜蜂活動風險指數
- 蜂箱 AI 建議文字
- 監測建議
- 是否啟動控制設備

## 判斷邏輯

- 高溫：啟動霧化器、遮陰與溫度警報
- 高濕：加強防潮與病害監測
- 強風：降低蜜蜂外勤活動判斷，加入風速警示
- 低溫：提醒蜂箱保溫並減少開箱

## 檔案說明

- `SKILL.md`：Skill 的正式工作流程與觸發說明
- `climate_rules.json`：台灣各地特殊氣候、代表座標、蜂場查詢、蜜蜂影響與監測建議資料
- `examples/`：屏東、宜蘭、台東三個示範輸出
